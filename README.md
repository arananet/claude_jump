# Claude Jump 🦀

A 2D pixel-art endless runner game inspired by the Chrome Dinosaur game, featuring the Anthropic Claude CLI logo!

## 🎮 Game Logic

The goal is to survive as long as possible by navigating Claude through an endless barrage of syntax errors and obstacles.

### Mechanics:
- **Controls**: Press `SPACE`, `ArrowUp`, or `TAP` the screen to jump.
- **Speed**: The game speed steadily increases the longer you survive.
- **Apples**: Collect floating apples for a massive **+50 point bonus**!

### Obstacles:
- **Ground Bugs**: Red syntax error bugs crawl along the ground. You must jump over them.
- **Flying Bugs (Score > 150)**: Red bugs with flapping wings. 
  - *Low flyers*: You must time a short hop over them.
  - *High flyers*: You have to run *under* them (jumping will get you killed).
- **Sinkholes (Score > 50)**: Gaps randomly appear in the ground track. If Claude's center of mass crosses the edge, he will plummet through the floor.
